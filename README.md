# saddle
A simple python library for finding saddle points of arbitrary stability index

# used libraries
* pytorch

# how to proceed
Simply look into example files:
* ex1_basic.py
* ex2_contour_plot.py
* ex3_benchmark.py
* ex4_gauss_random_parabolic.py

help files are provided under

```
help(find_saddle)
help(show_trajectory)
```
